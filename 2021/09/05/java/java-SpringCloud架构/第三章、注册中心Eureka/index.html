一、注册中心Eureka作用，解决了什么问题

1. 注册中心的作用：
  • 管理服务:服务太多需要管理，当服务多达成千上百时需要管理
  • 服务的依赖关系:服务之间是蜘蛛网的关联关系，需要确定管理依赖关系
  • 服务的注册
  • 服务的发现
  • 服务rest注册发现 负载均衡 故障转移的能力
2. 三种角色:
  • eureka server注册发现。
  • service provider 服务提供方，自己注册到eureka。
  • service consumer 服务消费方，从eureka得到注册列表，消费服务。
