一、为什么要用网管，他解决了什么问题

网关如同乐队的指挥员，调动全局

网关的职责：
  1. 统一入口：为全部微服务提供唯一入口点，网关起到外部和内部隔离，保障了后台服务的安全性。
  2. 鉴权校验：识别每个请求的权限，拒绝不符合要求的请求。
  3. 动态路由：动态的将请求路由到不同的后端集群中。
  4. 减少客户端与服务的耦合，服务可以独立发展，通过网关层来做映射。
