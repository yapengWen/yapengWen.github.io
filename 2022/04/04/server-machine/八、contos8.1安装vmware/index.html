二、下载VMWare Workstation Pro 16.2.3

VMWare Workstation的下载页面：https://www.vmware.com/in/products/workstation-pro.html
提示：可将下载链接复制到迅雷下载的更快

秘钥：YZ718-4REEQ-08DHQ-JNYQC-ZQRD0

如果失效可以百度搜索寻找秘钥

三、安装所需的所有依赖项

VMware Workstation Pro 16需要必要的构建工具来构建所需的VMware内核模块，通过在终端中运行以下命令，在CentOS 8中安装开发工具，如果在安装过程中提示，请按“y”：

sudo dnf -y groupinstall “Development Tools”

安装完成后，还需要另一个依赖项，称为elfutils-libelf-devel，这将促进VMware内核模块的构建，通过运行如下命令来安装它：

sudo dnf -y install elfutils-libelf-devel zlib-devel

四、安装和启动VMWare Workstation Pro 16.x的方法

sudo chmod +x VMware-Workstation-Full-16.2.3-15018445.x86_64.bundle
$ sudo ./VMware-Workstation-Full-16.2.3-15018445.x86_64.bundle

安装成功会出现下面提示：

  • 亲测此方法可以安装vmware 16.2.3版本，此方法貌似只能安装官网最新的那版
  • 16.2以上不支持虚拟机共享无法远程操作，需要远程操作请参考另一种方法：VMware ESXI 7.0及VCenter 7.0搭建
