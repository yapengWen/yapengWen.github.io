计算多边形不超出矩形范围的最大缩放比例

计算方法

  1. 获取矩形的width,height也就是x,y
  2. 循环多边形的每个坐标点，取到最大的xmax,ymax
  3. 计算scalex的缩放比例=xmax/x;计算scaley的缩放比例=ymax/y
  4. 最大缩放比例就是scalex和scaley中最小的比例
